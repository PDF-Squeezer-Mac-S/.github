## 📄 Overview

<a href="#pdfSqueezer">PDF Squeezer</a> is a PDF compression utility that reduces the file size of PDF documents through image recompression, color space optimization, and metadata removal — making PDFs smaller for email attachment, web upload, storage, and sharing without requiring manual optimization knowledge.

The <a href="#pdfSqueezer">drag-and-drop batch processing</a> accepts multiple PDF files simultaneously — dragging a folder of scanned documents, a set of client reports, or a collection of any PDF files onto PDF Squeezer and processing all of them in a single operation. Batch compression eliminates the need to process files one at a time, making it practical to optimize large collections of PDFs without individual attention to each file. The <a href="#pdfSqueezer">configurable compression quality settings</a> control the tradeoff between file size and visual quality — high compression for files where small size matters more than perfect image reproduction, low compression for files where image quality must be maintained, and a range of presets between the extremes for different use cases. The settings apply to the recompression of embedded images, which are typically the dominant contributor to PDF file size.

The <a href="#pdfSqueezer">JPEG image recompression</a> applies lossy compression to images embedded in PDFs — the scanned pages, photographs, and graphics that make PDF files large. By recompressing these images at a lower quality level appropriate to the document's viewing context rather than the original scan quality, PDF Squeezer achieves substantial file size reductions. A scanned document that doesn't need to be printed at full resolution for email sharing compresses significantly without quality loss apparent at normal viewing sizes. The <a href="#pdfSqueezer">grayscale conversion option</a> converts color PDF documents to grayscale — documents that contain color images but don't need color for their purpose (text-heavy documents with incidental color elements) reduce significantly through grayscale conversion since color information multiplies image data.

<a href="#pdfSqueezer">Metadata removal</a> strips embedded metadata, thumbnails, and other non-essential data from PDFs — reducing file size by removing data that doesn't affect the document's readable content. <a href="#pdfSqueezer">Color profile optimization</a> processes embedded ICC color profiles that add to PDF file size without affecting appearance in typical viewing contexts.

<a href="#pdfSqueezer">Custom compression profiles</a> save specific quality and setting combinations as named presets — defining a profile for email attachments, a profile for web upload, a profile for archiving, and applying the appropriate profile to each batch without reconfiguring settings each time. <a href="#pdfSqueezer">Folder monitoring</a> watches a specified folder and automatically compresses new PDFs that appear in it — enabling automatic processing of scanned documents and downloaded PDFs without manual operation for each incoming file.

PDF Squeezer addresses the gap between how PDFs are created and how they need to be used. Scanners produce PDFs at high resolution because storage is cheap and quality should be preserved — but these high-resolution scans are frequently far larger than necessary for the actual use case of emailing a document to a colleague or uploading a form to a website. PDF Squeezer bridges this gap by reducing file size to match the use context without requiring users to understand PDF compression parameters.

## 🙋 FAQ

<details>
<summary>Does PDF Squeezer support batch processing?</summary>

Yes. Multiple PDFs process simultaneously through drag-and-drop — entire folders compress in a single operation without individual file attention.

</details>

<details>
<summary>Does PDF Squeezer overwrite the original PDF?</summary>

PDF Squeezer can be configured to save compressed files to a separate output folder rather than overwriting originals.

</details>

<details>
<summary>How much does PDF Squeezer reduce file size?</summary>

Reduction varies with the content and compression settings. Image-heavy PDFs (scanned documents, photo-rich documents) typically compress significantly. Text-only PDFs with few or no images compress less dramatically.

</details>

<details>
<summary>Does PDF Squeezer support grayscale conversion?</summary>

Yes. Color PDFs convert to grayscale for additional size reduction when color is not needed for the document's purpose.

</details>

Yes. Folder monitoring watches a specified folder and automatically compresses new PDFs that appear in it.
